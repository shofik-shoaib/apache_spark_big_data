  •  abstract class BTypes extends AnyRef

    The BTypes component defines The BType class hierarchy.

    The BTypes component defines The BType class hierarchy. A BType stores all type information that is required after building the ASM nodes. This includes optimizations, generation of InnerClass attributes and generation of stack map frames.

    The representation is immutable and independent of the compiler data structures, hence it can be queried by concurrent threads.

  44.  final def typedOpcode(opcode: Int): Int

    Some JVM opcodes have typed variants.

    Some JVM opcodes have typed variants. This method returns the correct opcode according to the type.

    opcode

    A JVM instruction opcode. This opcode must be one of ILOAD, ISTORE, IALOAD, IASTORE, IADD, ISUB, IMUL, IDIV, IREM, INEG, ISHL, ISHR, IUSHR, IAND, IOR IXOR and IRETURN.

    returns

    The opcode adapted to this java type. For example, if this type is float and opcode is IRETURN, this method returns FRETURN.
